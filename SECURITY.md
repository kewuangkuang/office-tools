# 安全说明

## 处理敏感文件的建议

本工具主要在浏览器本地处理文件，但仍建议按下面方式使用：

1. 处理普通文件：可以直接使用在线页面。
2. 处理财务、银行、合同、证照等敏感文件：建议下载源码后本地打开 `index.html`。
3. 不要把真实敏感样例文件提交到 GitHub Issue、Pull Request 或公开讨论区。
4. 不要使用来历不明的二次部署版本处理敏感文件。

## OCR 注意事项

OCR 功能依赖第三方 CDN 和语言包。公司内网、断网或安全策略严格的环境下，OCR 可能无法加载。

## 报告安全问题

如果发现潜在安全问题，请提交不含敏感文件的 Issue，说明：

- 影响的功能
- 复现步骤
- 浏览器和操作系统
- 是否涉及 OCR / PDF / Excel

请不要上传真实财务资料、银行回单、身份证、营业执照等文件。
