# 更新日志

### 新增

- PDF 拆分结果支持选择导出格式：PDF 或 JPG 图片；多张 JPG 会自动打包成 ZIP。
- 接入飞书问题反馈表，网页顶部和首页底部都可以进入反馈入口。
- README 增加问题反馈入口。

- 隐私说明补充：反馈表中填写的内容、联系方式、截图或附件会提交到维护者的飞书多维表格，用于问题跟进和功能改进。

### 清理

- 清理已下线的 PDF 转 Excel 功能残留代码和说明，避免用户误以为该功能仍可用。
